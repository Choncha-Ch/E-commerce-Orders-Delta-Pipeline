# E-commerce-Orders-Delta-Pipeline
This repo contains a live demo of Databricks and Spark lessons Day 3 Week 7

## Theme
E-commerc Oders operations and fulfillments

## Main flow
- Landed files in a Unity Catalog volume
- Auto loader ingestion into Delta
- Delta table managment with UPDATE, DELETE and MERGE
- Delta history and time travel
- Incremental arrival of new files
